This repository currently includes scripts for simulating parent-offspring datasets with MZ and DZ twins. The code accounts for family relatedness. The grid-based power analysis enables users to examine whether indirect and direct effects achieve sufficient statistical power across a range of scenarios. The script also allows the results to be displayed in a visually appealing format.
