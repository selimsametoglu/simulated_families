This repository currently includes script/code to simulate parent-offspring dataset with MZ/DZ twins. The code accounts for the family relatedness. Grid type - power analysis allows for investigating whether indirect/direct effects achieve sufficient power for various scenerios. The script also allows for printing the results in a visually pleasing way.
